.. _nsml dataset ls:

List dataset
------------

    Usage:
        .. code-block:: console

            nsml dataset ls [Options]

        NSML에 등록된 모든 데이터셋의 이름, 크기, 올린 사람을 보여줍니다.

    Example:
        .. code-block:: console

            nsml dataset ls -j

    Options:
        .. py:attribute:: -j, --json-return

            json 형식으로 보여줍니다.
