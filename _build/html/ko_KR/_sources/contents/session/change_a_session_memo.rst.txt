.. _nsml memo:

Change a session memo
---------------------

    Usage:
        .. code-block:: console

            nsml memo [Options] SESSION MEMO

        현재 session 에 있는 memo/description 를 작성합니다. 이미 메모가 있다면 덮어씁니다.

    Example:
        .. code-block:: console

            nsml memo nsmlteam/NER/4 "First trial"
