.. _nsml model ls:

list models
-------------

    Usage:

        .. code-block:: console

            nsml model ls [Options] SESSION

        session에서 생성된 model을 확인합니다.

    Example:
        .. code-block:: console

            nsml model ls -j nsmlteam/NER/4

    Options:
        .. py:attribute:: -j, --json-return

            json format 으로 출력합니다.
